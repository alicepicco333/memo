Widoco Test - Meme Ontology

How to Use:

  1. Start a local HTTP server in this directory: python -m http.server 8000
  2. Visit: Widoco Online
  3. Upload or paste the URL: http://localhost:8000/meme_ontology_unpopulated.ttl
  4. Click "Generate Documentation"

Alternative: Direct Testing

Open browser console (F12) and check for CORS errors.

Ontology should have:

  • Base URI: http://www.semanticweb.org/meme-ontology
  • Element namespace: http://www.semanticweb.org/meme-ontology#
  • 144 classes with camelCase URIs
  • 28 object properties
  • 19 data properties
